# springboot-crud-api

This is for learning- which explain : How to implement a CRUD api 

## We need
- IDE.
- mysql database.
- postman.

## How to start
- Open STS or Eclipse
- Run the db. The db folder has a docker-copmose.yaml which run a db for development purpose
```
	cd db
	docker-compose up -d
```
- Build the project
	- Eclipse
	```
		Click on project -> maven -> Update Project
	```	
	- Commandline
	```
	  ./mvnw clean install
	``` 
- Run
```
./mvnw spring-boot:run
```
